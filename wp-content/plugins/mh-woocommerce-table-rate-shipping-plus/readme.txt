=== WooCommerce Table Rate Shipping Plus ===
Requires at least: 3.9
Tested up to: 4.1
Stable tag: 1.7.1

Support documentation is available at our support website: http://mangohour.com/support

== Installation ==

1. Transfer the 'mh-woocommerce-table-rate-shipping-plus' directory to the '/wp-content/plugins/' directory of your WordPress installation
2. Navigate to the WordPress admin area
3. Navigate to Plugins
4. Activate WooCommerce Table Rate Shipping Plus
